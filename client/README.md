# Next.js Starter Template

A clean, minimal Next.js starter optimized for quickly kicking off new projects.

## Quick Start

- Clone or copy this folder as your new project base.
- Update the project name in `package.json`.
- Install dependencies: `npm install`.
- Run the dev server: `npm run dev`.
- Open `http://localhost:3000` to view the app.

## Where to Edit

- Home page: `src/app/page.js`
- Global layout and metadata: `src/app/layout.js`
- Global styles: `src/app/globals.css`
- Static assets: `public/`

## Included

- Next.js App Router
- Tailwind CSS pre-configured
- Minimal layout, no analytics or SEO vendor scripts

## Recommended Next Steps

- Add routes under `src/app/<route>/page.js`
- Configure SEO metadata in `src/app/layout.js`
- Create UI components under `src/components/`
- Add environment variables via `.env.local` as needed

## Deploy

- Vercel: push to a repo and import at https://vercel.com
- Static host or custom server: see Next.js deployment docs
